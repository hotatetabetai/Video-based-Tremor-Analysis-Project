# Toward Markerless Video-based Tremor Analysis - Project Page

This repository contains the static project page for:

**Toward Markerless Video-based Tremor Analysis: Objective Quantification of Pathological Tremor in Mouse Preclinical Models**

The page is built from the Academic Project Page Template and is intended for GitHub Pages deployment.

## Links

- Project page: https://hotatetabetai.github.io/Video-based-Tremor-Analysis-Project/
- Code repository: https://github.com/hotatetabetai/Markerless-Video-based-Tremor-Analysis
- Paper PDF: `static/pdfs/paper.pdf`

## Local Preview

Open `index.html` directly in a browser, or serve the directory with a small static server:

```bash
python3 -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```

## Deployment

For GitHub Pages, enable Pages in the repository settings and deploy from the `main` branch root.
